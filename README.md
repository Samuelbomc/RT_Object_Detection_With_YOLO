# RT_Object_Detection_With_YOLO
El proyecto se basa en el uso del programa de Yolov11 para la realizacion y entrenamiento de un algortimo de deteccion de objetos (personas, carros y motos), para ello se hace uso de dos datasets proporcionados por el CETAD (Centro de Entrenamiento de Tripulaciones de Aeronaves de Defensa), uno de ellos realizado por las FAC (Fuerza Aeroespacial Colombiana) y otro dado por VisDrone, ambos con el proposito de capacitar el algortimo para conseguir resultados optimos menores al 50% de error segun la metrica mAP@ (Mean Average Precision)
Para este fin evaluaremos diferentes tipos de redes neuronales
